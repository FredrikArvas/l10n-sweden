# l10n-sweden — Swedish Odoo Localizations

Odoo 18 modules for Swedish-specific functionality.

## Modules

### l10n_se_ssn — Personnummer (SSN)

Adds a `personnummer` field to `res.partner` with full Luhn validation.

**Accepted formats:** `YYYYMMDD-NNNN`, `YYMMDD-NNNN`, with or without dash.  
**Validation:** Luhn algorithm per [Samlogic](https://www.samlogic.com/blogg/2012/11/validering-av-personnummer/).  
**Odoo version:** 18.0
